Welcome to Lumens

What is Lumens?

Lumens is a lightweight JavaScript slideshow which works without any dependencies like JQuery or other bloated libraries. It doesn't rely on a own CSS so you have nothing messing with your personal styling.

Lumens was built for developers. It doesn't add any navigation or pagination but gives you all the tools to add them yourself. This way you have full control over what happens, and when it does so.

It is completely written in TypeScript and commented thorouhly so if you want to change the behaviour even further than all the options already allow, you'll be there in no time.

Documentation

Properties

Lumens.container: HTMLElement The Element on which the slider was created
Lumens.wrapper: HTMLElement A DOM Element created by Lumens containing all slides and placed inside the container
Lumens.slides: HTMLCollection A HTMLCollection containing all the slides of the slider. This will also contain cloned slides if you enable the infinite loop option.
Lumens.options: Options The currently active options including not set, default values, based on the width of the window and the responsive settings.
Lumens.initialOptions: Options The options passed to the slider upon creation.
Lumens.currentPosX: number = 0 The current offset on the X-Axis
Lumens.currentPage: number = 0 The current slide that is shown. If there are more than one slide per page the first visible slide on the left is used as the current one. This value starts at the number zero
Lumens.logWarnings: boolean If this is set to true, the slideshow might log warnings or unpredictable behaviour.
Lumens.autoplayInterval: number The amount of milliseconds until the slideshow will continue to the next slide automatically. If set to zero, there will be no autoplay.

Options

Options.slidesPerPage?: number The amount of slides that should be displayed in a single page
Options.preventSelection?: boolean Will set 'userSelect' to none if set to true
Options.freeScroll?: boolean If set to true, the slideshow will no longer jump to the closest slide on stop drag
Options.animationSpeed?: number Amount of milliseconds it takes to animate to a page
Options.responsive?: ResponsiveObject[] Special options for certain screen widths
Options.loop?: boolean If set to true, the slider can be scrolled infinitly
Options.startingPage?: number The page the slideshow should start on
Options.autoplay?: number | false Amount of milliseconds until the slideshow goes to the next slide automatically
Options.draggable?: boolean If set to false, the slideshow will not be draggable with mouse- or touch-events
Options.preventTouchDrag?: boolean If set to true touch devices can not scroll left/right or up/down when touching the slider
Options.dragThreshold?: number The amount of pixels that have to be dragged to change the slide
Options.inheritOptions?: boolean If set to true, the responsive options that are not set will be inherited by the initial options, otherwise unset options will use the default
Options.arrowKeys?: boolean If set to true, the slider can be controlled using the left and right arrow key

Callbacks

onDragging Is called whenever the user drags the slideshow
onStopDragging Is called whenever the user stops dragging
onAnimating Is called when an animation starts
onFinishAnimating Is called when an animation is finished
onDestroy Is called when the slider is destoyed
onInit Is called when the slider finished initailizing
onChangeResponsive Is called when a responsive breakpoint is reached
onSlideChange Is called when the slide is starting to change
onSlideChanged Is called when the slide is changed
